📜 Shell Scripting Collection
This directory contains a collection of Bash shell scripts that automate common Linux administration and DevOps tasks.
These scripts were created as hands-on practice for Linux automation and demonstrate real-world DevOps use cases.

## Folder Description
### sample_scripts
Contains small scripts used for learning and practicing Linux Shell Scripting concepts.
Examples include:
- User Management
- Disk Monitoring
- Backup Automation
- Log Cleanup
- AWS Resource Tracking
- Server Health Check
- Network Connectivity Check
---
### system-health-report
A complete DevOps project that automates Linux server health monitoring and generates daily health reports.
Features include:
- CPU Usage
- Memory Usage
- Disk Usage
- Running Services
- Logged-in Users
- Top CPU Processes
- Top Memory Processes
- Daily Log Generation
- Cron Scheduling
